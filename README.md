# Online-Meal-Website
Responsive food delivery website using HTML & CSS
# 🍽️ MyOnlineMeal.com

A responsive food delivery website built with **HTML**, **CSS**, and a touch of responsive design for various devices. This is a simple frontend project created as part of my web development learning journey.

---

## 📌 Project Overview

**MyOnlineMeal.com** is a mock food delivery website that showcases a homepage, services section, client logos, and a contact form. The site is styled using custom CSS and is responsive for mobile and tablet devices using media queries.

---

## 🛠️ Technologies Used

- **HTML5** – Structure of the webpage  
- **CSS3** – Styling and layout  
- **Google Fonts** – Typography (`Bree Serif`)  
- **Responsive Design** – Mobile-friendly with media queries  
- **VS Code** – Development environment

---

## 📱 Features

- ✅ Attractive landing page with a welcoming message  
- ✅ Service cards with images and titles  
- ✅ Responsive navbar  
- ✅ Client logos section  
- ✅ Contact form for user input  
- ✅ Responsive across devices (uses `phone.css` for screens < 1156px)
